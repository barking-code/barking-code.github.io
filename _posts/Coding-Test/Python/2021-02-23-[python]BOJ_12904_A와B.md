[BOJ 12904](https://www.acmicpc.net/problem/12904)

```python
from sys import stdin

def solution():
    S = [*stdin.readline().strip()]
    T = [*stdin.readline().strip()]
    
    ls = len(S)
    lt = len(T)
    while lt > ls:
        if T[-1] == "A":
            T = T[:-1]
        else:
            T = T[:-1]
            T.reverse()
        lt -= 1

    return 1 if S == T else 0
print(solution())
```
